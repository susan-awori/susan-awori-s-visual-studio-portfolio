## Completed Projects

### 1. Cow, Grass & Lion – Strategy Game

A JavaScript-based strategy game that reimagines Rock-Paper-Scissors using biological interactions.

- Key Features: Circular predator-prey logic, custom UI console, and responsive interface.
- [Live Demo](https://susan-awori.github.io/Cow-Grass-Lion-Game/)
- [Source Code](https://github.com/susan-awori/Cow-Grass-Lion-Game.git)

### 2. ScanPay – Dynamic QR Solution

A UX case study and technical implementation of a mobile payment interface using transaction-specific QR codes.

- Key Features: Camera integration, mobile-first design, and simulated transaction feedback.
- [Live Demo](https://susan-awori.github.io/Dynamic-QR-UX-Case/)
- [Source Code](https://github.com/mebjas/html5-qrcode)

### 3. UzaLeo – Seller Onboarding

A high-performance landing page designed for seller onboarding with a focus on conversion and accessibility.

- Key Features: Semantic HTML5, high-contrast visual hierarchy, and optimized performance.
- [Live Demo](https://susan-awori.github.io/uza-leo-seller-onboarding-page/)
- [Source Code](https://github.com/susan-awori/uza-leo-seller-onboarding-page)

### 4. Interactive Calendar Application

A functional web application for navigating dates and managing timelines.

- Key Features: Dynamic date logic, month/year navigation, and fully responsive grid layout.
- [Live Demo](https://susan-awori.github.io/SUU-CALENDAR/)
- [Source Code](https://github.com/susan-awori/SUU-CALENDAR)
